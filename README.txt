este proyecto fue desarrollado mediante herramientas de programacion las cuales permiten su correcto funcionamiento
el proyecto desarrollado cuenta con algunas caracteristicas principales:

* el proyecto cuenta con un error al momento de actualizar un CONTENIDO no un APUNTE, el codigo fue revisado y no se encontró el error
* los URL ingresados deben ser de paginas publicas, de lo contrario en la prevista saldrá un rechazo de conexión
* la capacidad de almacenar, registrar y editar contenidos
* la capacidad de almacenar, registrar y editar apuntes
* cuenta con una prevista del URL ingresado en la sección apuntes la cual está también disponible al momento de editar el apunte
* cuenta con un home en el que se presentan los contenidos registrados
* cuenta con una pestaña "apuntes" en la tabla "crear contenido" la cual nos permite ingresar a los apuntes previamente registrados

este proyecto fue realizado bajo algunos lenguajes de programación como php,css y html. fue desarrollado en el ambiente
de programacion Visual Studio Code el cual facilitó un uso más dinamico del codigo fuente.

aclarar que este software esta en una version básica y para llevarlo a otro nivel requiere un trabajo más arduo.
